# Lojas Grátis

O guia completo para escolher a melhor plataforma de e-commerce no Brasil.

## Sobre o Projeto

**URL**: https://lojasgratis.com.br

Lojas Grátis é um ranking completo das melhores plataformas de e-commerce do Brasil. Comparamos custos, recursos, prós e contras para ajudar empreendedores a escolher a plataforma ideal para sua loja virtual.

## Tecnologias

Este projeto é construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento Local

Para trabalhar localmente, clone o repositório e execute:

```sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## Build e Deploy

```sh
# Criar build de produção
npm run build

# Visualizar build localmente
npm run preview
```

O deploy é feito automaticamente via GitHub Actions para a Hostinger.
